            \item \textcolor{MainBlue}{\textbf{(C)}} What does a near-zero ACF of $r_t$ suggest about predictability? What do the significant ACF values of $r_t^2$ indicate?
        \end{itemize}

\begin{frame}{Exercise 1: Solution}
    \begin{cminipage}{0.95\textwidth}
    \begin{exampleblock}{Solution}
        {\small
        \begin{itemize}\setlength{\itemsep}{3pt}
            \item \textcolor{MainBlue}{\textbf{(A)}} $P_t$ typically shows a non-stationary trending pattern; $r_t$ fluctuates around zero with periods of high and low volatility (volatility clustering).

Significant ACF($r_t^2$) $\Rightarrow$ \textbf{ARCH/GARCH effects}: volatility is predictable even when returns are not.
        \end{itemize}
        }
    \end{exampleblock}

\begin{frame}{Exercise 2: Raw Periodogram}
    \begin{cminipage}{0.95\textwidth}
    \begin{alertblock}{Question}
        Given $n = 256$ observations, the periodogram at Fourier frequency $\omega_j = 2\pi j/n$ is:
        \[
            I(\omega_j) = \frac{1}{2\pi n}\left|d(\omega_j)\right|^2, \qquad
            d(\omega_j) = \sum_{t=1}^{n} Y_t\, e^{-i\omega_j t}
        \]
    \end{alertblock}

    \vspace{0.15cm}

    \begin{block}{Exercise}
        \begin{itemize}\setlength{\itemsep}{3pt}
            \item \textcolor{MainBlue}{\textbf{(A)}} Plot $I(\omega_j)$ against $\omega_j \in [0, \pi]$. How do you identify the dominant frequency?
            \item \textcolor{MainBlue}{\textbf{(B)}} If the series has a \emph{quarterly} pattern (period $= 4$), at which angular frequency $\omega^*$ do you expect the largest periodogram peak?
            \item \textcolor{MainBlue}{\textbf{(C)}} What does a completely flat periodogram indicate?
        \end{itemize}
    \end{block}

    \vspace{0.15cm}

    \begin{center}
        \textit{Answer on next slide...}
    \end{center}
    \end{cminipage}
\end{frame}

\begin{frame}{Exercise 2: Solution}
    \begin{cminipage}{0.95\textwidth}
    \begin{exampleblock}{Solution}
        {\small
        \begin{itemize}\setlength{\itemsep}{3pt}
            \item \textcolor{MainBlue}{\textbf{(A)}} The dominant frequency is the $\omega_j$ at which $I(\omega_j)$ reaches its global maximum. The corresponding period is $T = 2\pi / \omega_j$ time units.
            \item \textcolor{MainBlue}{\textbf{(B)}} A quarterly pattern has period $T = 4$, so $\omega^* = 2\pi/4 = \pi/2 \approx 1.571$ rad. This is the 64th Fourier frequency for $n = 256$ (since $j^* = n/T = 64$).
            \item \textcolor{MainBlue}{\textbf{(C)}} A flat periodogram $I(\omega_j) \approx c$ for all $j$ indicates \textbf{white noise}: all frequencies contribute equally to the series variance ($\int_0^\pi f(\omega)\,d\omega = \gamma(0)$, where $f(\omega) = \sigma^2/(2\pi) = $ const).
        \end{itemize}
        }
    \end{exampleblock}
    \end{cminipage}
    \quantlet{TSA\_ch12\_periodogram}{https://github.com/danpele/TSA/tree/main/TSA_ch12/TSA_ch12_periodogram}

\begin{frame}{Exercise 3: Periodogram Inconsistency}
    \begin{cminipage}{0.95\textwidth}
    \begin{alertblock}{Question}
        Show that the periodogram is an \emph{inconsistent} estimator of the spectral density.
    \end{alertblock}

    \vspace{0.2cm}

    \begin{block}{Exercise}
        \begin{itemize}\setlength{\itemsep}{3pt}
            \item \textcolor{MainBlue}{\textbf{(A)}} State the asymptotic distribution of $I(\omega_j)$ as $n \to \infty$.
            \item \textcolor{MainBlue}{\textbf{(B)}} Derive $\text{Var}[I(\omega_j)]$ as a function of $n$. Does it go to zero?
            \item \textcolor{MainBlue}{\textbf{(C)}} Propose a solution that produces a consistent estimator, and explain the bias-variance tradeoff involved.
        \end{itemize}
    \end{block}

    \vspace{0.3cm}

    \begin{center}
        \textit{Answer on next slide...}
    \end{center}
    \end{cminipage}
\end{frame}

\begin{frame}{Exercise 3: Solution}
    \begin{cminipage}{0.95\textwidth}
    \begin{exampleblock}{Solution}
        {\footnotesize
        \begin{itemize}\setlength{\itemsep}{3pt}
            \item \textcolor{MainBlue}{\textbf{(A)}} For $\omega_j \neq 0, \pi$:
                $\dfrac{2\,I(\omega_j)}{f(\omega_j)} \xrightarrow{d} \chi^2_2 \;\Rightarrow\; I(\omega_j) \stackrel{d}{\approx} f(\omega_j)\cdot\chi^2_2/2$
            \item \textcolor{MainBlue}{\textbf{(B)}} Since $\chi^2_2$ has variance 4: $\text{Var}[I(\omega_j)] \approx f(\omega_j)^2 \cdot 4/4 = f(\omega_j)^2$. This does \textbf{not} decrease with $n$ $\Rightarrow$ periodogram is \textbf{inconsistent}.
            \item \textcolor{MainBlue}{\textbf{(C)}} \textbf{Smoothing} over $m$ adjacent ordinates:
                $\hat{f}(\omega_j) = \frac{1}{2m+1}\sum_{k=-m}^{m} I(\omega_{j+k})$.
                Variance $\propto 1/m$, bandwidth $\propto m/n$. More smoothing $\Rightarrow$ lower variance but higher bias.
        \end{itemize}
        }
    \end{exampleblock}
    \end{cminipage}
    \quantlet{TSA\_ch12\_periodogram}{https://github.com/danpele/TSA/tree/main/TSA_ch12/TSA_ch12_periodogram}
\end{frame}

    \begin{block}{Exercise}

            \item \textcolor{MainBlue}{\textbf{(A)}} Derive the spectral density $f(\omega)$ for $\omega \in [0, \pi]$.
            \item \textcolor{MainBlue}{\textbf{(B)}} Compute the ratio $f(0)/f(\pi)$. Interpret the result.
            \item \textcolor{MainBlue}{\textbf{(C)}} What is a \emph{red spectrum}? Give an economic example.
        \end{itemize}
    \end{block}

    \vspace{0.3cm}

    \begin{center}
        \textit{Answer on next slide...}
    \end{center}
    \end{cminipage}
\end{frame}

\begin{frame}{Exercise 4: Solution}
    \begin{cminipage}{0.95\textwidth}
    \begin{exampleblock}{Solution}
        {\small
        \begin{itemize}\setlength{\itemsep}{3pt}
            \item \textcolor{MainBlue}{\textbf{(A)}} For AR(1) with $\phi = 0.8$:
                $f(\omega) = \frac{1}{2\pi}\cdot\frac{1}{1.64 - 1.6\cos\omega}$
            \item \textcolor{MainBlue}{\textbf{(B)}} $f(0) = \frac{25}{2\pi}$, $f(\pi) = \frac{1}{2\pi \cdot 3.24}$, so
                $\frac{f(0)}{f(\pi)} = \left(\frac{1+\phi}{1-\phi}\right)^2 = \left(\frac{1.8}{0.2}\right)^2 = 81$.
                Most variance is concentrated at low frequencies.
            \item \textcolor{MainBlue}{\textbf{(C)}} A \textbf{red spectrum} is dominated by low frequencies: most variance is at slow oscillations (long cycles). GDP, consumption, and stock prices all exhibit red spectra --- they move slowly and trend persistently, with little energy at high frequencies.
        \end{itemize}
        }
    \end{exampleblock}
    \end{cminipage}

\begin{frame}{Exercise 5: Welch vs.\ Multitaper Estimation}
    \begin{cminipage}{0.95\textwidth}
    \begin{alertblock}{Question}
        Compare two spectral estimators on the same series of $n = 512$ observations:
        \textbf{Welch}: $L = 4$ overlapping segments; \quad \textbf{Multitaper}: $NW = 4$, $K = 7$ DPSS tapers.
    \end{alertblock}

    \vspace{0.2cm}

    \begin{block}{Exercise}
        \begin{itemize}\setlength{\itemsep}{3pt}
            \item \textcolor{MainBlue}{\textbf{(A)}} Compute the frequency resolution (bandwidth) for each estimator.
            \item \textcolor{MainBlue}{\textbf{(B)}} Which estimator has lower variance? Why?
            \item \textcolor{MainBlue}{\textbf{(C)}} Describe the bias-variance tradeoff for each method.
        \end{itemize}
    \end{block}

    \vspace{0.3cm}

    \begin{center}
        \textit{Answer on next slide...}
    \end{center}
    \end{cminipage}
\end{frame}

\begin{frame}{Exercise 5: Solution}
    \begin{cminipage}{0.95\textwidth}
    \begin{exampleblock}{Solution}
        {\footnotesize
        \begin{itemize}\setlength{\itemsep}{3pt}
            \item \textcolor{MainBlue}{\textbf{(A)}} Welch ($L=4$, $n_{seg}=128$): $\Delta\omega_{\text{Welch}} \approx 2/128 \approx 0.0156$ cyc/obs.
                Multitaper ($NW=4$, $n=512$): $\Delta\omega_{\text{MT}} \approx 2NW/n = 8/512 = 0.0156$ cyc/obs.
                Both have the same frequency resolution for these parameters.
            \item \textcolor{MainBlue}{\textbf{(B)}} Multitaper averages $K = 7$ independent eigen-spectra $\Rightarrow$ variance $\propto 1/K$. For equal bandwidth, multitaper has lower variance: equivalent degrees of freedom $\approx 2K = 14$ vs $\approx 2L = 8$.
            \item \textcolor{MainBlue}{\textbf{(C)}} Both methods trade frequency resolution for variance reduction. Multitaper achieves near-optimal sidelobe suppression via DPSS tapers, minimising spectral leakage --- preferred for most applications.
        \end{itemize}
        }
    \end{exampleblock}
    \end{cminipage}

\begin{frame}{Exercise 7: HP Filter Transfer Function}
    \begin{cminipage}{0.95\textwidth}
    {\small
    \begin{alertblock}{Question}
        The HP filter extracts trend $\tau_t$ by minimising
        $\sum(Y_t-\tau_t)^2 + \lambda\sum(\Delta^2\tau_t)^2$.
        The squared gain of the \emph{cycle} component is:
        $|A_{HP}(\omega)|^2 = \frac{\lambda\,[2(1-\cos\omega)]^2}{1 + \lambda\,[2(1-\cos\omega)]^2}$
    \end{alertblock}

    \vspace{0.15cm}

    \begin{block}{Exercise}
        \begin{itemize}\setlength{\itemsep}{3pt}
            \item \textcolor{MainBlue}{\textbf{(A)}} Evaluate $|A_{HP}(\omega)|^2$ at $\omega = 0$ and $\omega = \pi$ for any $\lambda$.
            \item \textcolor{MainBlue}{\textbf{(B)}} Find the half-power cutoff frequency for $\lambda = 1600$.
            \item \textcolor{MainBlue}{\textbf{(C)}} Summarise Hamilton (2018)'s critique of the HP filter.
        \end{itemize}
    \end{block}

    \vspace{0.15cm}

    \begin{center}
        \textit{Answer on next slide...}
    \end{center}
    }
    \end{cminipage}
\end{frame}

\begin{frame}{Exercise 7: Solution}
    \begin{cminipage}{0.95\textwidth}
    \begin{exampleblock}{Solution}
        {\footnotesize
        \begin{itemize}\setlength{\itemsep}{3pt}
            \item \textcolor{MainBlue}{\textbf{(A)}} At $\omega=0$: $2(1-\cos 0)=0 \Rightarrow |A_{HP}(0)|^2=0$ (trend removed).
                At $\omega=\pi$: $|A_{HP}(\pi)|^2 = 16\lambda/(1+16\lambda)\approx 1$ (high freq.\ passed).
            \item \textcolor{MainBlue}{\textbf{(B)}} Setting $|A_{HP}(\omega)|^2=0.5$: $\lambda[2(1-\cos\omega)]^2=1$, so
                $2(1-\cos\omega)=1600^{-1/2}=0.025$, giving $\omega^*\approx 0.158$ rad $\approx 2\pi/40$ (period $\approx 40$ quarters $= 10$ years).
            \item \textcolor{MainBlue}{\textbf{(C)}} \textbf{Hamilton (2018)}: HP filtering is equivalent to a regression on lagged values. The resulting cycle is spuriously correlated at all lags and imposes ARMA(4,4) structure on white noise --- generating \emph{artificial} business cycles. Hamilton recommends a simple 2-step regression alternative.
        \end{itemize}
        }
    \end{exampleblock}
    \end{cminipage}

\begin{frame}{Exercise 8: GPH Estimator}
    \begin{cminipage}{0.95\textwidth}
    {\small
    \begin{alertblock}{Question}
        For ARFIMA(0,$d$,0), the GPH estimator fits OLS to:
        \[
            \log I(\omega_j) = c - d\log[4\sin^2(\omega_j/2)] + u_j, \quad j=1,\ldots,m
        \]
    \end{alertblock}

    \vspace{0.15cm}

    \begin{block}{Exercise}
        \begin{itemize}\setlength{\itemsep}{3pt}
            \item \textcolor{MainBlue}{\textbf{(A)}} What is the OLS estimator $\hat{d}$? How is it computed?
            \item \textcolor{MainBlue}{\textbf{(B)}} What bandwidth $m$ should be used? Theoretical recommendations?
            \item \textcolor{MainBlue}{\textbf{(C)}} What is the asymptotic standard error of $\hat{d}$?
        \end{itemize}
    \end{block}

    \vspace{0.15cm}

    \begin{center}
        \textit{Answer on next slide...}
    \end{center}
    }
    \end{cminipage}
\end{frame}

\begin{frame}{Exercise 8: Solution}
    \begin{cminipage}{0.95\textwidth}
    \begin{exampleblock}{Solution}
        {\footnotesize
        \begin{itemize}\setlength{\itemsep}{3pt}
            \item \textcolor{MainBlue}{\textbf{(A)}} Let $x_j = -\log[4\sin^2(\omega_j/2)]$ and $y_j = \log I(\omega_j)$:
                $\hat{d} = \frac{\sum_{j=1}^m (x_j - \bar{x})(y_j - \bar{y})}{\sum_{j=1}^m (x_j - \bar{x})^2}$ \quad (OLS slope)
            \item \textcolor{MainBlue}{\textbf{(B)}} Theory: $m = O(n^\alpha)$, $\alpha \in (0,1)$. Common practice: $m \approx n^{0.5}$ to $n^{0.6}$. Too small $m$ $\Rightarrow$ high variance; too large $m$ $\Rightarrow$ bias from short-memory contamination.
            \item \textcolor{MainBlue}{\textbf{(C)}} $\text{SE}(\hat{d}) = \pi/\sqrt{6m}$. For $n=1000$, $m=32$: SE $\approx 0.226$. Hurst: $H = d + 0.5$; persistent series have $d>0$ ($H>0.5$).
        \end{itemize}
        }
    \end{exampleblock}
    \end{cminipage}

\begin{frame}{Exercise 9: Local Whittle Estimator}

    \begin{alertblock}{Question}

    \begin{block}{Exercise}
        \begin{itemize}\setlength{\itemsep}{3pt}
            \item \textcolor{MainBlue}{\textbf{(A)}} Write the Local Whittle objective function and state what it minimises.
            \item \textcolor{MainBlue}{\textbf{(B)}} What bandwidth $m$ is used, and what is the asymptotic variance of $\hat{d}_{LW}$?
            \item \textcolor{MainBlue}{\textbf{(C)}} Why is Local Whittle more efficient than GPH?
        \end{itemize}
    \end{block}

    \vspace{0.3cm}

    \begin{center}
        \textit{Answer on next slide...}
    \end{center}
    \end{cminipage}

\begin{frame}{Exercise 9: Solution}
    \begin{cminipage}{0.95\textwidth}
    \begin{exampleblock}{Solution}
        {\footnotesize
        \begin{itemize}\setlength{\itemsep}{3pt}
            \item \textcolor{MainBlue}{\textbf{(A)}} Local Whittle minimises the restricted Whittle log-likelihood:
                \[
                    Q(d) = \log G(d) - 2d\,\bar{\lambda}_m, \quad G(d) = \frac{1}{m}\sum_{j=1}^{m}\frac{I(\omega_j)}{\omega_j^{-2d}}, \quad \hat{d}_{LW} = \arg\min_d Q(d)
                \]
            \item \textcolor{MainBlue}{\textbf{(B)}} Bandwidth $m=O(n^{0.6\text{--}0.8})$; for $n=1000$ typically $m\approx 100$--$250$. Asymptotic variance: $\text{Var}(\hat{d}_{LW})=1/(4m)$. For $m=100$: SD $=0.05$.
            \item \textcolor{MainBlue}{\textbf{(C)}} GPH uses OLS on log-periodogram (heavy-tailed $\chi^2_2$ errors); LW uses a proper likelihood approximation. LW is asymptotically efficient: $\text{Var}=1/(4m)$ vs GPH's $\approx 1.645/m$ for the same $m$.
        \end{itemize}
        }
    \end{exampleblock}
    \end{cminipage}

\begin{frame}{Exercise 10: Cross-Spectral Analysis}
    \begin{cminipage}{0.95\textwidth}
    \begin{alertblock}{Question}
        Suppose $Y_t = X_{t-4} + \varepsilon_t$ where $\varepsilon_t \perp X_s$ for all $s, t$.
    \end{alertblock}

    \vspace{0.2cm}

    \begin{block}{Exercise}
        \begin{itemize}\setlength{\itemsep}{3pt}
            \item \textcolor{MainBlue}{\textbf{(A)}} Derive the cross-spectrum $S_{XY}(\omega) = e^{-4i\omega}S_{XX}(\omega)$. What is the phase function $\phi(\omega)$?
            \item \textcolor{MainBlue}{\textbf{(B)}} Compute the squared coherence $C^2_{XY}(\omega)$: (i) without noise, (ii) with noise $\varepsilon_t \sim (0, \sigma^2_\varepsilon)$.
            \item \textcolor{MainBlue}{\textbf{(C)}} What does the phase slope $-d\phi/d\omega$ tell you, and what is its value here?
        \end{itemize}
    \end{block}

    \vspace{0.2cm}

    \begin{center}
        \textit{Answer on next slide...}
    \end{center}
    \end{cminipage}
\end{frame}

\begin{frame}{Exercise 10: Solution}
    \begin{cminipage}{0.95\textwidth}
    \begin{exampleblock}{Solution}
        {\small
        \begin{itemize}\setlength{\itemsep}{3pt}
            \item \textcolor{MainBlue}{\textbf{(A)}} Transfer function $H(\omega) = e^{-4i\omega}$, so $S_{XY}(\omega) = e^{-4i\omega}\,S_{XX}(\omega)$.
                The phase is $\phi(\omega) = -4\omega$ (linear in $\omega$).
            \item \textcolor{MainBlue}{\textbf{(B)}} $C^2_{XY}(\omega) = |S_{XY}|^2/(S_{XX} \cdot S_{YY})$:
                (i) No noise: $S_{YY} = S_{XX} \Rightarrow C^2_{XY}(\omega) = 1$ (perfect linear relationship).
                (ii) With noise: $C^2_{XY}(\omega) = S_{XX}(\omega)/[S_{XX}(\omega) + \sigma^2_\varepsilon/(2\pi)] < 1$.
            \item \textcolor{MainBlue}{\textbf{(C)}} The \textbf{group delay} $= -d\phi/d\omega = 4$ periods at ALL frequencies. A constant group delay confirms the system introduces a \textbf{pure shift of 4 periods} with no phase distortion.
        \end{itemize}
        }
    \end{exampleblock}
    \end{cminipage}

\section{Discussion Topics}

\begin{frame}{Discussion: Choosing the Right Spectral Estimator}
    \begin{cminipage}{0.95\textwidth}
    {\small
    \begin{alertblock}{Key Question}
        You have $n = 1000$ observations. Which spectral estimator do you use and why?
    \end{alertblock}
    \begin{block}{Decision Framework}
        \begin{enumerate}\setlength{\itemsep}{0pt}
            \item \textbf{Need a quick look at dominant frequencies?}
                \begin{itemize}\setlength{\itemsep}{0pt}
                    \item Raw periodogram --- fast but noisy
                \end{itemize}
            \item \textbf{Need low variance with good sidelobe suppression?}
                \begin{itemize}\setlength{\itemsep}{0pt}
                    \item Multitaper (DPSS) --- the modern standard
                \end{itemize}
            \item \textbf{Short series or nonstationary segments?}
                \begin{itemize}\setlength{\itemsep}{0pt}
                    \item Welch (overlapping segments) --- reduces sensitivity to nonstationarity
                \end{itemize}
            \item \textbf{Series known to follow an AR/ARMA model?}
                \begin{itemize}\setlength{\itemsep}{0pt}
                    \item Parametric AR spectrum --- smooth and model-consistent
                \end{itemize}
        \end{enumerate}
    \end{block}
    }
    \end{cminipage}

\begin{frame}{Discussion: Interpreting Economic Spectra}
    \begin{cminipage}{0.95\textwidth}
    \vspace{-0.2cm}
    {\small
    \begin{alertblock}{Key Question}
        You compute the spectral density of quarterly GDP growth and find most power at frequencies below $\pi/16$. What does this imply?
    \end{alertblock}
    \begin{block}{Interpretation Guide}
        \begin{itemize}\setlength{\itemsep}{0pt}
            \item Frequency $\omega < \pi/16$ $\Rightarrow$ period $> 32$ quarters $= 8$ years: \textbf{long cycles}
            \item The spectrum is \textbf{red}: persistent, slowly-moving component dominates
            \item Business cycle band: $\omega \in [\pi/16, \pi/3]$ (periods 6--32 quarters)
        \end{itemize}
    \end{block}
    \begin{exampleblock}{Practical Rules}
        \begin{itemize}\setlength{\itemsep}{0pt}
            \item \textbf{Trend}: power near $\omega \approx 0$ (very low frequency)
            \item \textbf{Business cycle}: power at 6--32 quarter periods (Baxter-King band)
            \item \textbf{Seasonal}: sharp peaks at $\omega = \pi/2$ (quarterly), $2\pi/12$ (monthly)
            \item \textbf{White noise}: flat spectrum across all frequencies
        \end{itemize}
    \end{exampleblock}
    }
    \end{cminipage}
    \quantlet{TSA\_ch12\_gdp}{https://github.com/danpele/TSA/tree/main/TSA_ch12/TSA_ch12_gdp}
\end{frame}

\begin{frame}{Discussion: HP Filter in Practice}
    \begin{cminipage}{0.95\textwidth}
    {\small
    \begin{alertblock}{Key Question}
        Should economists stop using the HP filter? What are the alternatives?
    \end{alertblock}
    \begin{block}{Arguments Against (Hamilton 2018)}
        \begin{itemize}\setlength{\itemsep}{0pt}
            \item Imposes spurious ARMA(4,4) structure on white noise
            \item Generates artificial cycles not present in the data
            \item End-of-sample estimates are unreliable
            \item Results are not invariant to the choice of $\lambda$
        \end{itemize}
    \end{block}
    \begin{exampleblock}{Alternatives}
        \begin{itemize}\setlength{\itemsep}{0pt}
            \item \textbf{Hamilton filter}: regress $Y_{t+h}$ on $Y_t, Y_{t-1}, Y_{t-2}, Y_{t-3}$; use residual as cycle
            \item \textbf{Bandpass filter} (Baxter-King, Christiano-Fitzgerald): isolate business-cycle frequencies directly
            \item \textbf{Unobserved components model}: explicitly model trend + cycle with state-space methods
        \end{itemize}
    \end{exampleblock}
    }
    \end{cminipage}
    \quantlet{TSA\_ch12\_filters}{https://github.com/danpele/TSA/tree/main/TSA_ch12/TSA_ch12_filters}
\end{frame}

\begin{frame}{Take-Home Exercises}
    \begin{cminipage}{0.95\textwidth}
    {\footnotesize
    \begin{enumerate}\setlength{\itemsep}{2pt}
        \item \textbf{Theoretical}: Show that for white noise $\varepsilon_t \sim \mathrm{WN}(0,\sigma^2)$, the spectral density is $f(\omega) = \sigma^2/(2\pi)$ for all $\omega \in [0,\pi]$.

        \item \textbf{Computation}: For the AR(2) process $Y_t = 0.6Y_{t-1} - 0.3Y_{t-2} + \varepsilon_t$:
            \begin{itemize}\setlength{\itemsep}{0pt}
                \item Derive the spectral density $f(\omega)$ analytically
                \item Simulate $n = 512$ observations and compare the periodogram with the theoretical spectrum
                \item Apply Welch and multitaper estimation; compare all three
            \end{itemize}

        \item \textbf{Applied}: Download monthly sunspot numbers (available at \texttt{sidc.be}):
            \begin{itemize}\setlength{\itemsep}{0pt}
                \item Estimate the spectral density and identify the $\approx 11$-year cycle
                \item Test for long memory; report $\hat{d}$ via GPH and Local Whittle
            \end{itemize}

        \item \textbf{Critical Thinking}: Why does applying the HP filter with $\lambda = 1600$ to monthly data (instead of quarterly) produce misleading results?
    \end{enumerate}
    }
    \end{cminipage}
    \quantlet{TSA\_ch12\_sunspot}{https://github.com/danpele/TSA/tree/main/TSA_ch12/TSA_ch12_sunspot}
\end{frame}

\begin{frame}{Exercise Solution Hints}
    \begin{cminipage}{0.95\textwidth}
    \vspace{-0.3cm}
    {\footnotesize
    \begin{block}{Hints}
        \begin{enumerate}\setlength{\itemsep}{1pt}
            \item By the Wiener-Khinchin theorem, $f(\omega) = \frac{1}{2\pi}\sum_{h=-\infty}^{\infty}\gamma(h)e^{-i\omega h}$. For WN: $\gamma(0)=\sigma^2$, $\gamma(h)=0$ for $h\neq 0$, so $f(\omega) = \sigma^2/(2\pi)$ --- flat.

            \item AR(2) spectrum: $f(\omega) = \frac{\sigma^2}{2\pi|1 - 0.6e^{-i\omega} + 0.3e^{-2i\omega}|^2}$. The process has a spectral peak (complex roots). Verify numerically: the peak should lie between 0 and $\pi/2$.

            \item Sunspot cycle: expect a sharp peak near $\omega = 2\pi/(11 \times 12) \approx 0.048$ rad/month. Long-memory tests typically give $\hat{d} \approx 0.3$--$0.5$ for sunspot numbers.

            \item HP filter calibration: $\lambda = 1600$ is designed for quarterly data. For monthly data, the correct value is $\lambda \approx 1600 \times 3^4 = 129{,}600$. Using $\lambda=1600$ on monthly data cuts off at the wrong frequency, distorting the extracted cycle.
        \end{enumerate}
    \end{block}
    }
    \end{cminipage}
