# TSA Chapter 0 Seminar: Exercise Solutions

This folder contains Python solutions for Seminar 0 exercises.

## Exercise Solutions

| File | Exercise | Topic |
|------|----------|-------|
| `TSA_ch0_ex1_ses.py` | Exercise 1 | Simple Exponential Smoothing with Î± = 0.3 |
| `TSA_ch0_ex2_metrics.py` | Exercise 2 | Forecast Error Metrics (MAE, MSE, RMSE, MAPE) |
| `TSA_ch0_ex3_seasonal.py` | Exercise 3 | Seasonal Indices and Deseasonalization |

## Features

Each solution file includes:
- **Step-by-step calculations** with explanations
- **Formulas** clearly shown
- **Visualizations** saved to `../../charts/`
- **Final answers** summary
- **Interpretation** of results

## Requirements

```bash
pip install numpy pandas matplotlib
```

## Usage

Run any exercise solution:

```bash
python TSA_ch0_ex1_ses.py
```

## Learning Objectives

1. **Exercise 1 (SES)**: Apply the SES formula iteratively, calculate forecast errors
2. **Exercise 2 (Metrics)**: Compare different error metrics, understand their properties
3. **Exercise 3 (Seasonal)**: Work with multiplicative seasonal indices, deseasonalize data
