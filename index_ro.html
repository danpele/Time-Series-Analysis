Formule Cheie

Descompunere

Aditivă: $X_t = T_t + S_t + \varepsilon_t$

Multiplicativă: $X_t = T_t \times S_t \times \varepsilon_t$

Staționaritate

$\mathbb{E}[X_t] = \mu$ (constantă)

$\text{Var}(X_t) = \sigma^2$ (constantă)

$\text{Cov}(X_t, X_{t+h}) = \gamma(h)$

Model AR(p)

$X_t = c + \sum_{i=1}^{p}\phi_i X_{t-i} + \varepsilon_t$

Staționară dacă rădăcinile în afara cercului unitate

Model MA(q)

$X_t = \mu + \sum_{j=1}^{q}\theta_j\varepsilon_{t-j} + \varepsilon_t$

Întotdeauna staționară

ARIMA(p,d,q)

$\phi(L)(1-L)^d X_t = \theta(L)\varepsilon_t$

$d$ = ordinul diferențierii

SARIMA

$(p,d,q) \times (P,D,Q)_s$

ARIMA sezonier + non-sezonier

Model VAR(p)

$\mathbf{Y}_t = \mathbf{c} + \sum_{i=1}^{p}\mathbf{A}_i\mathbf{Y}_{t-i} + \boldsymbol{\varepsilon}_t$

Extensie multivariată a AR

Cauzalitate Granger

$X \rightarrow Y$: trecutul lui $X$ prezice $Y$

$H_0$: coeficienții lui $X$ = 0

GARCH(1,1)

$\sigma_t^2 = \omega + \alpha \varepsilon_{t-1}^2 + \beta \sigma_{t-1}^2$

Model pentru volatilitate

Value at Risk

$\text{VaR}_\alpha = \mu + z_\alpha \cdot \sigma_t$

Măsură de risc la nivel $\alpha$

Cointegrare

$Y_t = \beta X_t + u_t$, $u_t \sim I(0)$

Relație de echilibru pe termen lung

Selecția Modelului

AIC $= -2\log L + 2k$

BIC $= -2\log L + k\log n$

Întrebarea 1: Descompunerea Seriilor de Timp

Când ar trebui folosită descompunerea aditivă în locul celei multiplicative?

  • A) Când amplitudinea sezonieră crește odată cu nivelul seriei
  • B) Când amplitudinea sezonieră rămâne constantă în timp
  • C) Când seria are o tendință descendentă puternică
  • D) Când seria conține valori lipsă
✓ Corect! Descompunerea aditivă se folosește când amplitudinea sezonieră rămâne constantă indiferent de nivelul seriei. Descompunerea multiplicativă este adecvată când sezonalitatea se scalează proporțional cu nivelul (amplitudinea crește sau scade cu trendul).
✗ Incorect. Răspunsul corect este B. Descompunerea aditivă se folosește când amplitudinea sezonieră rămâne constantă în timp, în timp ce descompunerea multiplicativă este adecvată când sezonalitatea se scalează cu nivelul seriei.

Întrebarea 2: Metodele de Netezire Exponențială

Care metodă de netezire exponențială ar trebui folosită pentru o serie cu atât tendință cât și sezonalitate?

  • A) Netezirea Exponențială Simplă (SES)
  • B) Metoda Holt-Winters
  • C) Media Mobilă Simplă
  • D) Metoda Liniară Holt
✓ Corect! Holt-Winters (Netezirea Exponențială Triplă) este concepută special pentru serii cu atât tendință cât și sezonalitate, folosind trei componente de netezire: una pentru nivel, una pentru tendință și una pentru sezonalitate.
✗ Incorect. Răspunsul corect este B. Holt-Winters gestionează atât tendința cât și sezonalitatea. SES este doar pentru serii staționare, Holt este pentru tendință fără sezonalitate, iar mediile mobile sunt simple și nu sunt adaptative.

De ce validarea încrucișată K-fold standard este problematică pentru datele de tip serie de timp?

  • C) Nu funcționează cu date sezoniere
  • D) Este prea lentă computațional
✓ Corect! K-fold standard încalcă ordinea temporală prin folosirea potențială a datelor viitoare pentru antrenare în timp ce testează pe trecut. Validarea încrucișată pentru serii de timp (fereastră glisantă sau în expansiune) menține ordinea temporală, antrenând întotdeauna pe trecut și testând pe viitor.
✗ Incorect. Răspunsul corect este B. Problema principală este că K-fold încalcă ordinea temporală, folosind potențial observații viitoare pentru antrenare. Aceasta creează scurgeri de informație și supraevaluează performanța modelului.

Ce se întâmplă când parametrul de netezire α crește către 1 în Netezirea Exponențială Simplă?

  • C) Previziunea ignoră toate observațiile trecute
  • D) Modelul devine mai complex
✓ Corect! Alpha mare (aproape de 1) pune mai multă pondere pe observațiile recente, făcând previziunea mai reactivă la schimbări dar și mai volatilă. Alpha mic netezește mai mult, punând mai multă pondere pe istoric.
✗ Incorect. Răspunsul corect este B. Pe măsură ce α crește către 1, mai multă pondere este pusă pe observațiile cele mai recente, făcând previziunea mai reactivă. La α=1, previziunea devine pur și simplu ultima observație (metodă naivă).

  • C) Teoretic nelimitat (practic constrâns de date/calcul)
  • D) Doar una, precum SARIMA
Corect! TBATS poate gestiona orice număr de perioade sezoniere prin termeni Fourier aditivi. Complexitatea practică limitează acest lucru.
Incorect. Răspunsul corect este C. TBATS poate teoretic gestiona oricâte perioade sezoniere.

Întrebarea 17: Sezonalitate Personalizată

Care model permite mai ușor adăugarea de componente de sezonalitate personalizate?

Corect! Funcția add_seasonality() a Prophet face adăugarea de sezonalități personalizate (ex: bi-lunară, trimestru fiscal) foarte simplă.

Întrebarea 18: Date Lipsă

Întrebarea 19: Eficiența Computațională

Care model este de obicei mai rapid pentru a se ajusta pe serii lungi cu sezonalitate înaltă?

Capitolul 5: Modele de Volatilitate ARCH/GARCH

Întrebarea 1: Volatility Clustering

Ce reprezintă „volatility clustering"?

  • B) Perioadele de volatilitate ridicată sunt urmate de perioade de volatilitate ridicată
  • C) Randamentele sunt corelate în timp
  • D) Distribuția randamentelor este normală
Corect! Volatility clustering = perioade agitate urmate de perioade agitate.
Incorect. Răspunsul corect este B. Volatilitatea tinde să persiste.

Întrebarea 2: Parametrii GARCH

În GARCH(1,1): σₜ² = ω + α·εₜ₋₁² + β·σₜ₋₁², ce reprezintă α?

  • A) Persistența volatilității
  • B) Nivelul de bază al volatilității
  • C) Reacția la șocuri recente (news coefficient)
  • D) Varianța necondiționată
Corect! α măsoară cât de mult reacționează volatilitatea la șocuri noi.
Incorect. Răspunsul corect este C. α = reacția la news.

Întrebarea 3: Condiția de Stationaritate

Care este condiția de stationaritate pentru GARCH(1,1)?

  • A) ω > 0
  • B) α + β = 1
  • C) α + β < 1
  • D) α > β
Corect! Pentru stationaritate, α + β < 1 (persistența totală sub 1).
Incorect. Răspunsul corect este C. Stationaritate: α + β < 1.

Întrebarea 4: Varianța Necondiționată

Care este formula varianței necondiționate în GARCH(1,1)?

  • A) σ̄² = ω
  • B) σ̄² = ω / (1 - α)
  • C) σ̄² = ω / (1 - α - β)
  • D) σ̄² = ω / (α + β)
Corect! Varianța necondiționată = ω / (1 - α - β).
Incorect. Răspunsul corect este C.

Întrebarea 5: Leverage Effect

Ce este „leverage effect"?

  • A) Șocurile pozitive cresc volatilitatea mai mult
  • B) Șocurile negative cresc volatilitatea mai mult decât cele pozitive
  • C) Volatilitatea este independentă de semnul șocurilor
  • D) Randamentele sunt asimetrice
Corect! Scăderile de preț măresc volatilitatea mai mult decât creșterile.
Incorect. Răspunsul corect este B. Leverage = asimetrie negativă.

Întrebarea 6: EGARCH

În EGARCH, un parametru γ negativ indică:

  • A) Absența leverage effect
  • B) Prezența leverage effect
  • C) Volatilitate constantă
  • D) Model nestationar
Corect! γ < 0 în EGARCH confirmă leverage effect.
Incorect. Răspunsul corect este B.

Întrebarea 7: Avantajul EGARCH

Care este principalul avantaj al EGARCH față de GARCH?

  • A) Este mai rapid de estimat
  • B) Nu necesită restricții de non-negativitate
  • C) Are mai puțini parametri
  • D) Este mai ușor de interpretat
Corect! EGARCH modelează ln(σ²), deci σ² > 0 automat.
Incorect. Răspunsul corect este B.

Întrebarea 8: Testul ARCH-LM

Ce test folosim pentru a detecta efecte ARCH în reziduuri?

  • B) Testul Ljung-Box pe reziduuri
  • C) Testul Engle (ARCH-LM)
  • D) Testul Breusch-Pagan
Corect! ARCH-LM testează heteroscedasticitatea condiționată.
Incorect. Răspunsul corect este C.

Întrebarea 9: Valori Tipice

Pentru S&P 500, valorile tipice ale α + β în GARCH(1,1) sunt:

  • B) 0.70 - 0.85
  • C) 0.95 - 0.99
  • D) Mai mare decât 1
Corect! Volatilitatea financiară este foarte persistentă (α + β ≈ 0.97-0.99).

Întrebarea 10: Distribuția Inovațiilor

Care distribuție este cel mai des folosită pentru inovațiile GARCH pentru a captura cozile groase?

Întrebarea 12: GARCH vs ARCH

Întrebarea 13: Value at Risk (VaR)

Reziduurile standardizate într-un model GARCH sunt:

Capitolul 10: Recapitulare Comprehensivă - De la Date la Prognoză

Întrebarea 1: Fluxul de Analiză

Care este ordinea corectă a fluxului de analiză a seriilor de timp?

  • A) Estimarea modelului, Explorarea datelor, Diagnostice, Prognoză
  • B) Explorarea datelor, Testarea staționarității, Selectarea modelului, Diagnostice, Prognoză
  • C) Prognoză, Explorarea datelor, Estimarea modelului, Diagnostice
  • D) Diagnostice, Selectarea modelului, Explorarea datelor, Prognoză
Corect! Fluxul sistematic: Explorare date → Testare staționaritate → Selectare model → Estimare → Diagnostice → Prognoză.
Incorect. Răspunsul corect este B. Întotdeauna se începe cu explorarea datelor înainte de estimarea modelului.

Întrebarea 2: RMSE vs MAE

Dacă RMSE este mult mai mare decât MAE, aceasta sugerează:

  • A) Modelul este supraajustat (overfitting)
  • B) Există unele erori mari aberante
  • C) Modelul este subajustat (underfitting)
  • D) Datele sunt staționare
Corect! RMSE penalizează mai mult erorile mari (pătrate). Un raport RMSE/MAE mare indică erori de prognoză foarte mari.
Incorect. Răspunsul corect este B. RMSE este mai sensibil la erorile mari decât MAE.

Întrebarea 3: Limitarea MAPE

MAPE este problematic atunci când:

  • A) Datele au o tendință
  • B) Valorile reale sunt aproape de sau egale cu zero
  • C) Orizontul de prognoză este lung
  • D) Se compară mai multe modele
Corect! MAPE = $\frac{100}{n}\sum|\frac{y_t - \hat{y}_t}{y_t}|$. Împărțirea la $y_t$ mic produce valori extreme sau nedefinite.
Incorect. Răspunsul corect este B. MAPE împarte la valorile reale, cauzând probleme aproape de zero.

Întrebarea 4: Gruparea Volatilității

Dacă randamentele S&P 500 prezintă perioade de volatilitate ridicată urmate de volatilitate ridicată, ar trebui să considerați:

  • A) ARIMA de ordin superior
  • B) Netezire exponențială
  • C) Model GARCH pentru varianță
  • D) Diferențiere sezonieră
Corect! Gruparea volatilității este o caracteristică cheie a randamentelor financiare. GARCH: $\sigma_t^2 = \omega + \alpha \varepsilon_{t-1}^2 + \beta \sigma_{t-1}^2$.
Incorect. Răspunsul corect este C. Modelele GARCH captează volatilitatea variabilă în timp.

Întrebarea 5: ADF vs KPSS

Dacă testul ADF nu respinge și KPSS respinge, seria este probabil:

  • A) Staționară
  • B) Nestaționară
  • C) Staționară cu tendință
  • D) Zgomot alb
Corect! ADF H0: rădăcină unitară. KPSS H0: staționaritate. Ambele sugerează nestaționaritate când ADF nu respinge și KPSS respinge.
Incorect. Răspunsul corect este B. Folosiți ambele teste împreună pentru confirmare.

Întrebarea 6: Sezonalitate Multiplicativă

Datele despre pasagerii aerieni arată o amplitudine sezonieră crescătoare în timp. Ce descompunere este potrivită?

  • A) Aditivă: $Y_t = T_t + S_t + R_t$
  • B) Multiplicativă: $Y_t = T_t \times S_t \times R_t$
  • C) Ambele funcționează la fel de bine
  • D) Niciuna---folosiți diferențierea
Corect! Când amplitudinea sezonieră se scalează cu nivelul, folosiți multiplicativă. Transformarea log convertește la aditivă.
Incorect. Răspunsul corect este B. Multiplicativă pentru efecte sezoniere proporționale.

Întrebarea 7: SARIMA pentru Date Lunare

Pentru date lunare cu sezonalitate anuală, perioada sezonieră $m$ în SARIMA este:

  • A) 4
  • B) 7
  • C) 12

Incorect. Răspunsul corect este C. Datele lunare au 12 observații pe an.

Întrebarea 8: Compararea Modelelor

La compararea prognozelor SARIMA și Prophet, care metrică este independentă de scală?

  • A) RMSE
  • B) MAE
  • C) MAPE
  • D) MSE
Corect! MAPE exprimă eroarea ca procent. Utilă pentru compararea între diferite scale/seturi de date.
Incorect. Răspunsul corect este C. MAPE este bazată pe procente și independentă de scală.

Întrebarea 9: Rupturi Structurale

Vânzările din SUA au suferit o ruptură structurală în timpul COVID-19. Prophet gestionează aceasta prin:

  • A) Diferențiere automată
  • B) Detectarea punctelor de schimbare în tendință
  • C) Ajustare sezonieră
  • D) Modelare GARCH
Corect! Punctele de schimbare ale Prophet permit pantei tendinței să se modifice în puncte specifice, capturând rupturi structurale precum COVID-19.
Incorect. Răspunsul corect este B. Punctele de schimbare modelează schimbările de regim în tendință.

Întrebarea 10: Testul Ljung-Box

După estimarea unui model ARIMA, testul Ljung-Box pe reziduuri testează pentru:

  • A) Normalitate
  • B) Autocorelație reziduală
  • C) Heteroscedasticitate
  • D) Staționaritate
Corect! Ljung-Box testează dacă reziduurile sunt zgomot alb (fără autocorelație). Respinge = model inadecvat.
Incorect. Răspunsul corect este B. Testează dacă autocorelațiile reziduale sunt conjunct zero.

Întrebarea 11: Interpretarea ACF/PACF

ACF scade exponențial și PACF se taie după lag-ul 2. Aceasta sugerează:

  • A) MA(2)
  • B) AR(2)
  • C) ARMA(2,2)
  • D) Mers aleatoriu
Corect! Tăierea PACF → ordinul AR. Tăierea ACF → ordinul MA. ACF descrescătoare + PACF tăiată la 2 = AR(2).
Incorect. Răspunsul corect este B. PACF determină ordinul AR.

Întrebarea 12: Randamente Logaritmice

Pentru S&P 500, folosim randamente logaritmice $r_t = \ln(P_t/P_{t-1})$ în loc de prețuri deoarece:

  • A) Prețurile sunt întotdeauna staționare
  • B) Randamentele sunt aproximativ staționare; prețurile nu sunt
  • C) Randamentele logaritmice sunt mai ușor de calculat
  • D) Randamentele au autocorelație mai puternică
Corect! Prețurile urmează un mers aleatoriu (nestaționare). Randamentele sunt (aproximativ) staționare și potrivite pentru ARIMA/GARCH.
Incorect. Răspunsul corect este B. Staționaritatea este necesară pentru majoritatea modelelor de serii de timp.

Întrebarea 13: AIC vs BIC

Comparativ cu AIC, BIC selectează de obicei:

  • B) Modele mai simple (mai parcimonioase)

Incorect. Răspunsul corect este B. Penalizarea mai puternică a BIC favorizează modele mai simple.

Întrebarea 14: Validare Încrucișată

Pentru validarea încrucișată a seriilor de timp, folosim:

Incorect. Răspunsul corect este C. Împărțirile aleatorii violează ordinea temporală.

Întrebarea 15: Sezonalitate Multiplă

Datele orare cu pattern-uri zilnice, săptămânale și anuale sunt cel mai bine gestionate de:

Întrebarea 16: Persistența GARCH

În GARCH(1,1), persistența ridicată a volatilității înseamnă că $\alpha + \beta$ este:

  • B) Aproape de 1
  • C) Mai mare decât 1
  • D) Negativ

Întrebarea 17: Incertitudinea Prognozei

Pe măsură ce orizontul de prognoză crește, intervalele de predicție de obicei:

  • B) Rămân constante

Întrebarea 18: Acuratețea Direcției

  • B) Mai rău decât ghicitul aleatoriu

Întrebarea 19: Principiul Selecției Modelului
